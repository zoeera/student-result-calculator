# student-result-calculator
A command-line Python program that takes student names and marks, then calculates class statistics automatically.
## What it does
- Accepts unlimited student entries
- stops when you type 'done'
- Calculates class average
- Finds highest and lowest scoring student
- Shows total number of students
## What I learned building this
- while loops with break conditions
- Lists of dictionaries
- Tracking highest and lowest values with if conditions
- float() for decimal marks
- round() for clean average display
## How to run it
1. Make sure Python is installed
2. Download 'result_calculator.py'
3. Open terminal and run:
python result_calculator.py
